# movies-backend
- NodeJS Version: 16.14.0
- npm Version: 8.3.1

---

### How to run project:
- Clone the project
- Run `npm install`
- Run `npm install ts-node` --> Only when you are getting this error: `'ts-node' is not recognized as an internal or external command.`
- Run `npm run start`

- Open `http://localhost:13000` in your browser
